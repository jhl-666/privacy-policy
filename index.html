欢迎使用 {{APP_NAME}}（以下简称“本应用”）。我们非常重视用户隐私与个人信息保护。请在使用本应用前仔细阅读并确认您理解本隐私政策。

1. 我们收集的信息

  • 设备与运行信息：例如设备型号、操作系统版本、唯一设备标识符（不包含明确身份证明信息）、日志信息等，用于崩溃修复与性能优化。
  • 应用使用数据：功能使用统计、页面浏览、操作日志（用于产品改进与分析）。
  • 用户提供的信息：例如您主动上传的照片、联系方式或填写的表单内容，仅在您明确授权或需要时收集。

2. 目的与使用

我们仅在下列合理必要的范围内使用上述信息：

  • 提供、维护和改进本应用的功能；
  • 对上传的照片进行处理（如证件照换底、裁剪等），并返回结果；
  • 分析并统计应用使用情况，以优化用户体验；
  • 在法律要求或为保护我们的权利时使用。

3. 共享与第三方

我们不会将您的个人信息出售给任何第三方。为实现服务，我们可能会与第三方服务提供商共享必要的信息，例如云存储、图像处理或统计分析服务。这些第三方仅可在为我们提供服务的范围内处理数据，并须遵守保密义务。

4. 数据保存与删除

我们会在实现数据收集目的所需的最短时间内保存您的数据；当您请求删除或停止使用服务后，我们将在合理期限内删除相关个人数据（法律另有规定的除外）。

5. 安全

我们已采取合理的技术与组织措施保护数据安全，包括传输过程中使用 HTTPS 加密和对敏感数据的访问控制。但请注意，任何传输网络均无绝对安全，我们无法完全保证数据 100% 不被非法访问。

6. 儿童隐私

本应用不面向 13 岁以下儿童。如果我们知悉收到来自该年龄段儿童的个人信息，我们将采取措施尽快删除相关数据。

  • 访问权：您可以请求访问我们保存的与您相关的信息；
  • 更正权：您可以请求更正不正确或不完整的信息；
  • 删除权：在合理情况下，您可请求删除您的个人信息；
  • 撤回同意：在您已给予同意的情况下，您可随时撤回。

8. 第三方 SDK 与 Cookie

本应用可能使用第三方 SDK（例如用于统计或一键登录的 SDK）。这些 SDK 的隐私政策与数据处理由相应第三方负责，建议您查阅其隐私条款。

我们可能会不时更新本隐私政策。若有重大变更，我们将在应用内或通过其他合理方式通知您，并在本页面更新生效日期。
